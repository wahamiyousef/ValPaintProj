# ValPaintProj
VALORANT Paint Software
Made with python

Features: 
  Pencil,
  Erase,  
  Paint Bucket, 
  Trash, 
  Spray Paint, 
  Shapes (line, square, circle, triangle [with option to fill the shape]),
  Horizontal/Vertical image flip,
  Crop tool,
  8 unique stamps
  4 different backgrounds,
  Brush colour and size adjustable,
